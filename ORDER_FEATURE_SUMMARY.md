## 🔧 技术实现细节

### **订单号生成算法**
```java
// 格式：YYYYMMDDHHMMSS + 6位随机数
String timeStr = LocalDateTime.now().format(DateTimeFormatter.ofPattern("yyyyMMddHHmmss"));
int randomNum = random.nextInt(900000) + 100000; // 确保6位数
return timeStr + randomNum;
```

### **订单创建流程**
1. **验证用户登录状态**
2. **生成唯一订单号**
3. **计算订单总金额**
4. **检查商品库存**
5. **创建订单记录**
6. **创建订单项记录**
7. **更新商品库存和销量**
8. **返回订单信息**

### **前端数据流**
1. **商品详情页** → 立即购买 → 订单确认页
2. **购物车页面** → 去结算 → 订单确认页
3. **订单确认页** → 提交订单 → 后端API → 订单创建成功

## 📊 数据库表结构

### **订单表 (order)**
- `order_id`: 订单ID (主键)
- `order_no`: 订单号 (唯一)
- `user_id`: 用户ID
- `total_amount`: 订单总金额
- `pay_amount`: 实付金额
- `order_status`: 订单状态 (1-待付款)
- `pay_status`: 支付状态 (0-未支付)
- `receiver_name`: 收货人姓名
- `receiver_phone`: 收货人电话
- `receiver_address`: 收货地址
- `remark`: 订单备注

### **订单项表 (order_item)**
- `item_id`: 订单项ID (主键)
- `order_id`: 订单ID (外键)
- `product_id`: 商品ID
- `product_name`: 商品名称
- `product_price`: 商品价格
- `quantity`: 购买数量
- `total_price`: 小计金额
- `spec`: 商品规格
- `color`: 商品颜色

## 🚀 使用说明

### **立即购买流程**
1. 在商品详情页选择规格和颜色
2. 点击"立即购买"按钮
3. 填写收货信息
4. 选择支付方式
5. 确认订单信息
6. 点击"确认支付"提交订单

### **购物车结算流程**
1. 在购物车页面选择要结算的商品
2. 点击"去结算"按钮

4. 选择支付方式
