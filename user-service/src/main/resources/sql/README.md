# 数据库脚本说明

## 订单表创建脚本

### 执行步骤：

1. 连接到你的MySQL数据库
2. 执行 `order_table.sql` 脚本

```sql
-- 在MySQL中执行以下命令
source /path/to/order_table.sql;
```

或者直接在MySQL客户端中复制粘贴SQL内容执行。

### 创建的表：

1. **order** - 订单主表
   - 包含订单基本信息：订单号、用户ID、金额、状态等
   - 包含收货信息：收货人、电话、地址等
   - 包含时间信息：创建时间、支付时间、发货时间等

2. **order_item** - 订单商品详情表
   - 包含订单中的商品信息：商品ID、名称、价格、数量等
   - 包含商品规格和颜色信息

### 测试数据：

脚本中包含了4个测试订单和对应的商品详情，用于测试订单管理功能。

### 订单状态说明：

- **订单状态 (order_status)**：
  - 1: 待付款
  - 2: 已付款  
  - 3: 已发货
  - 4: 已完成
  - 5: 已取消

- **支付状态 (pay_status)**：
  - 0: 未支付
  - 1: 已支付

- **发货状态 (delivery_status)**：
  - 0: 未发货
  - 1: 已发货

### 注意事项：

- 确保数据库用户有创建表的权限
- 如果表已存在，脚本会跳过创建（使用 IF NOT EXISTS）
- 测试数据可以删除，不影响系统正常运行
