# 项目经历

## 微服务电商平台（全屋定制）

**项目时间：** 2024.10-2024.12

### 功能实现

1. 负责全屋定制模块开发，实现定制案例展示、设计师管理和预约系统，支持多图片上传和文件管理，通过数据库查询优化（索引、分页、字段筛选）提升列表加载效率40%。

2. 设计订单状态管理机制，确保订单流程一致性；集成评价晒单功能，使用Redis ZSet实现热门搜索关键词排序，通过缓存机制提升并发读写性能50%。

### 技术栈

- Java (Spring Boot 3)
- Spring Security + JWT (无状态认证)
- MyBatis-Plus (动态SQL)
- MySQL (事务隔离级别优化)
- Redis (ZSet排序、缓存穿透防护)
- React + React Router (前端框架)
- UmiJS + Ant Design Pro (管理端)

### 成果

- 系统稳定支持100+并发用户操作，日均订单处理量超过200单。
